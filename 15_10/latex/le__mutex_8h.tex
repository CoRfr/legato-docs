Declare a static mutex reference variable and accessor functions.

This is handy when you need a single, file-\/scope mutex for use inside your module to protect other file-\/scope data structures from multi-\/threaded race conditions.

Adding the line 
\begin{DoxyCode}
1 LE\_MUTEX\_DECLARE\_REF(MyMutexRef);
\end{DoxyCode}
 near the top of your file will create a file-\/scope variable called \char`\"{}\+My\+Mutex\+Ref\char`\"{} of type le\+\_\+mutex\+\_\+\+Ref\+\_\+t and functions called \char`\"{}\+Lock\char`\"{} and \char`\"{}\+Unlock\char`\"{} that access that variable.

See \hyperlink{c_mutex_c_mutex_locking_tips}{Tip} for more information.
